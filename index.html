Project Three

Explore Travel

For this five day group project, we built a full-stack application to plan and display a round-the-world trip. Users can browse the famous sites that are included in our database and add them to their dashboard. They can then visualise their trip on their dashboard's map and change the order of their journey using a react drag-and-drop feature. I focussed on the backend for this project whilst my teammates worked on the frontend and the dashboard component respectively.

Software:

React, JavaScript, HTML, CSS, Bulma, MongoDB, Mongoose, Express, Axios, Mocha, Chai, Webpack.

Project Four

Sunset Barlevard

In this week-long group project we created an app which enabled users to create and display their own bar crawls. We spent the first day building out the backend together so we would all have a good understanding of the database structure, then went on to tackle different areas of the frontend, all tracking our progress on a joint Trello board.We included a feature where anyone visiting the site can filter the crawls to include only those with predominently outside spaces- perfect for soaking up the rays on those sunny days.

Software:

React, JavaScript, HTML, CSS, Bulma, PostgreSQL, SQLAlchemy, Flask, Sinon, Enzyme, Mocha, Chai, Webpack.
